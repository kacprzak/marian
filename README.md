Marian
======

2d platformer game. Another Super Mario clone, but with multiplayer. Just for fun.


Dependencies
------------

* boost
* SDL2
* SDL2_image
* OpenGL
* Box2D
* Lua
* CEGUI
* zlib


Third-party code
----------------

* RapidXml
* base64


Tools
-----

* [Tiled Map Editor](http://www.mapeditor.org/ "Tiled")
